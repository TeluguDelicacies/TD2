# Telugu Delicacies - Image Requirements

This document outlines all image requirements for the Telugu Delicacies website, including aspect ratios, file names, locations, and formats.

## Brand Assets

### Logo
- **Required Aspect Ratio**: 1:1 (Square)
- **File Name**: `logo.png`
- **File Location**: `./images/`
- **File Format**: PNG (with transparency)
- **Recommended Size**: 256x256px minimum

### Hero Background
- **Required Aspect Ratio**: 16:9 or 2:1 (Landscape)
- **File Name**: `hero-background.jpg`
- **File Location**: `./images/`
- **File Format**: JPG
- **Recommended Size**: 1920x1080px minimum

## CSS Implementation Notes

### Product Showcase (Homepage)
- Images are displayed at **120px height** with `object-fit: cover`
- Standing pouch effect applied with `drop-shadow(0 8px 16px rgba(0, 0, 0, 0.15))`

### Shop Page Product Cards
- Images are displayed at **180px height** with `object-fit: cover`
- Hover scaling effect: `transform: scale(1.02)`

### Category Cards
- Images are displayed at **200px height** with `object-fit: cover`
- Border radius: `12px`

---

## Image Optimization Recommendations

1. **File Size**: Keep images under 500KB for optimal loading
2. **Compression**: Use 85-90% JPEG quality for best balance
3. **Format**: Use JPG for photos, PNG for graphics with transparency
4. **Responsive**: Provide high-resolution images (2x the display size minimum)
5. **Alt Text**: All images should have descriptive alt text for accessibility
